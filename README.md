This project is developed as part of the technical evaluation.  
It recreates a modern SaaS homepage (Header, Footer, and Homepage) using core PHP with reusable components.

## ⚙️ How to Run the Project (XAMPP)

### 1. Install XAMPP
Download and install XAMPP: https://www.apachefriends.org/index.html

### 2. Start Apache
Open XAMPP Control Panel and start **Apache**

### 3. Move Project to htdocs
Copy the project folder into:

C:\xampp\htdocs\

Example:
C:\xampp\htdocs\dbass_task\

### 4. Run in Browser
Open your browser and go to:

http://localhost/dbass_task/
